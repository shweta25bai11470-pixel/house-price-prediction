# House Price Prediction using Machine Learning

##  Overview

This project predicts house prices based on features like square foot, number of bedrooms (BHK), and location coordinates.

##  Technologies Used

* Python
* Pandas
* Scikit-learn

##  Features Used

* Square feet (SQUARE_FT)
* Number of BHK (BHK_NO)
* Construction status
* RERA approval
* Location (Latitude & Longitude)
* Property type (BHK/RK)
* Seller type (Dealer/Owner/Builder)

## Model Used

Linear Regression

## Results

* Mean Absolute Error (MAE): ~131
* R² Score: ~0.40

## How to Run

1. Install dependencies:
   pip install pandas scikit-learn

2. Run the code:
   python house-price-prediction.py

## Features

* Predicts house price based on user input
* Interactive terminal-based input system
* Displays model performance metrics
